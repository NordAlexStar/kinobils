# KINO NOX — 1. un 2. sprints

Blazor Server lietotne kinoteātra repertuāra un seansu izvēlei.

## Palaist

```powershell
cd .\src\KinoRezervacija.Web
dotnet run
```

Atveriet konsolē norādīto adresi. Demonstrācijas dati tiek glabāti lietotnes atmiņā, tādēļ pēc pārstartēšanas tiek atjaunoti sākotnējie dati.

## Iekļauts

- filmu katalogs ar žanra un “Šodien” filtriem;
- filmas detalizētā informācija un seansu saraksts;
- zāles plāns ar brīvajām/aizņemtajām vietām;
- operatora forma filmu un seansu pievienošanai;
- konta reģistrācija un autorizācija ar paroles jaucējvērtību;
- viesu pirkums vai pirkums no reģistrēta konta;
- vietu izvēle, aizņemtas vietas aizsardzība un pasūtījuma apstiprināšana;
- digitāla biļete ar unikālu QR kodu demonstrācijas formā;
- vienību testi kataloga datiem.

> Maksājums un e-pasta nosūtīšana šajā mācību laidienā ir simulēti. Dati tiek glabāti atmiņā un pēc lietotnes pārstartēšanas tiek dzēsti.
